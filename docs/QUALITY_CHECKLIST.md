# Quality Checklist

يجب أن يمر أي GPT بهذه القائمة قبل النشر.

## Checklist

- [ ] System Prompt مكتمل.
- [ ] Conversation Starters جاهزة.
- [ ] Knowledge تمت مراجعته (إن وجد).
- [ ] 20 حالة اختبار على الأقل.
- [ ] مراجعة اللغة.
- [ ] مراجعة الروابط.
- [ ] مراجعة صفحة المنتج.
- [ ] مراجعة بيانات المتجر.
- [ ] جاهزية للنشر.

## Review Criteria

### System Prompt

- واضح
- متخصص
- خالٍ من التناقضات
- مناسب لنوع GPT

### Conversation Starters

- قصيرة
- عملية
- مرتبطة بالاستخدام الحقيقي

### Knowledge

- لا يحتوي على معلومات قديمة
- مرتبط بالمنتج فقط
- منظم وسهل المراجعة

### Tests

- يغطي الاستخدامات الأساسية
- يتحقق من جودة الإخراج
- يكشف الأخطاء الشائعة

### Language

- اللغة سليمة
- النبرة ثابتة
- الأسلوب مناسب للجمهور

### Links

- تعمل الروابط
- لا توجد روابط وهمية غير مقصودة
- صفحة المنتج تشير إلى الوجهة الصحيحة

### Store Data

- الاسم صحيح
- slug صحيح
- الحالة صحيحة
- الفئة صحيحة

## Launch Rule

لا يُعتمد أي GPT للإطلاق إلا إذا تم تعليم جميع العناصر الأساسية في هذه القائمة أو تسجيل سبب واضح لأي عنصر غير مكتمل.
