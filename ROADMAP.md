# Roadmap

خطة تطوير **LAFRYHI GPT Store** عبر 3 مراحل.

## 1) MVP

الهدف:

- إطلاق سريع ومباشر
- عرض GPTs كبطاقات واضحة
- صفحة تفاصيل لكل GPT
- ملف بيانات واحد `gpts.json`
- صفحات About و Contact

مخرجات هذه المرحلة:

- موقع ثابت بسيط
- قابل للتشغيل المحلي
- قابل للنشر على Netlify أو Vercel أو GitHub Pages

## 2) النسخة المحسنة

الهدف:

- تحسين تجربة التصفح والاكتشاف
- رفع جودة العرض التسويقي
- إضافة بنية محتوى أقوى

أفكار التطوير:

- إضافة صور فعلية أو أيقونات أفضل لكل GPT
- تحسين SEO للصفحات
- إضافة أقسام مثل Featured و New و Coming Soon
- تحسين البحث والفلترة
- إضافة صفحة FAQ
- إضافة نموذج تواصل فعلي
- إضافة إحصاءات بسيطة أو عدادات استخدام
- دعم لغتين إذا احتجت ذلك لاحقًا

## 3) النسخة الاحترافية المدفوعة أو العامة

الهدف:

- تحويل الموقع إلى منتج عام أو واجهة بيع احترافية
- دعم التوسّع مع نمو عدد GPTs والأدوات

مسارات محتملة:

### إذا كانت النسخة عامة

- تحسين الهوية البصرية
- كتابة صفحات تسويقية قوية
- إضافة صفحات هبوط منفصلة لكل فئة
- تحسين الأداء وبيانات الميتا
- إضافة محتوى منظم للمشاريع والأدوات

### إذا كانت النسخة مدفوعة

- إضافة نظام اشتراك أو صلاحيات وصول
- ربط الدفع
- إضافة لوحة إدارة للمحتوى
- إظهار بعض الأدوات كمدفوعة أو محجوبة
- ربط النشر بنظام تحديث أسهل من JSON اليدوي

## ترتيب التنفيذ المقترح

1. تثبيت MVP الحالي
2. تحسين المحتوى والهوية
3. إضافة إدارة محتوى أفضل
4. تحديد نموذج العمل: عام أو مدفوع
5. تنفيذ الميزات المتقدمة فقط بعد وضوح الاتجاه التجاري
